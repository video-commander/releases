## 🚀 What is Video Commander?

**Video Commander** is a fast, offline desktop application designed for professional video workflows.

It focuses on **deep media inspection and delivery tooling**, combining low-level file analysis with modern encoding and packaging workflows — all without relying on cloud services.

### Core capabilities

- Inspect MP4 / ISOBMFF structure and metadata
- Visualize box trees and file layout
- Explore tracks, samples, and timing data
- Preview media assets
- Convert and package media using FFmpeg-based pipelines
- Designed for developers, video engineers, and technical operators

Video Commander is built as a **native desktop app** (macOS first), prioritizing performance, transparency, and control.
